Fix Fill The Letter / Build The Word

Current problem:
Children cannot choose incorrect letters.

Fix:
Allow all answer options to be selected.

Example:

Word:
c _ t

Options:
a
o
i
u

If child picks:
o

Result:
Incorrect

Explanation:
The word is cat. The missing sound is a.

Move to next question after feedback.

Do not force the correct answer.

When any letter is tapped:

* Speak the phonics sound immediately

Example:

Child taps:
s

Voice says:
sss

NOT:
ess

Apply this to:

* Fill The Letter
* Build The Word
* Missing Letter games

Review history must save:

* Word shown
* Options shown
* Child answer
* Correct answer
* Correct/incorrect
* Explanation
