3. Fix confusing Yes/No questions

Some games, especially Health & Food, ask questions like:

“Is this healthy?”

But the answers are:

* Yes
* No
* Maybe
* Sometimes

This is confusing.

For direct Yes/No questions, only show:

* Yes
* No

Do not include Maybe or Sometimes unless the question is specifically designed to teach “sometimes food”.

Rules:

* If the question asks “Is X healthy?” → answers must only be Yes / No
* If the question asks “Is X good for every day?” → answers must only be Yes / No
* If the question asks “Should you brush your teeth?” → answers must only be Yes / No

Only use “Sometimes” when the wording clearly supports it, for example:
“Is cake an everyday food or a sometimes food?”

Then answers can be:

* Everyday food
* Sometimes food

Audit all Health & Food games and any other games with Yes/No style questions.

Make questions and answer options simple, clear, age-appropriate, and not confusing.
