"Manage Points"

Adult-only feature (requires passcode).

Allow:

* Add Points
* Deduct Points
* Reset Points

Example:

+10
+50
+100

-10
-50
-100

or custom amount.

Changes must:

* Save immediately
* Persist after app restart
* Sync with rewards shop
* Sync with pet progress
* Sync with child statistics

Add an audit log:

"Point History"

Example:

Parent added +50 points
Teacher deducted -20 points
Reward purchased -100 points

Include:

* Date
* Time
* Action
* Amount

Current problem:

This gives away the answer.

ONLY.

### Missing Letter Game

Current:

L I _ N

is read as:

"L I underscore N"

Change speech behaviour.

Underscores should be spoken as:

"Blank"

Example:

L I _ N

should be read:

"L I Blank N"

C A _

should be read:

"C A Blank"

Never say "underscore".
