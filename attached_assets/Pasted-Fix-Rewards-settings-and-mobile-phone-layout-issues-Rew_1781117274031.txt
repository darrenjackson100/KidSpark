Fix Rewards settings and mobile phone layout issues.

Rewards & Shop Settings:

When a parent or teacher creates a new child profile:

* Rewards & Shop must be OFF by default
* Points card should be hidden unless Rewards & Shop is enabled
* Shop should not appear anywhere unless enabled
* Parents/teachers can enable it later by editing the child profile

Reward Requests:

Children should only be able to request rewards if:

* Rewards Shop is enabled
* The reward is active
* They have enough points

If they do not have enough points:

* Do not allow the request
* Do not create a pending request
* Show:
  "You need more points to unlock this reward."

Allow children to:

* Cancel/delete their own pending reward requests

Allow parents/teachers to:

* Approve requests
* Deny requests

Approving:

* Deduct points
* Add entry to reward history

Denying:

* Do not deduct points
* Add entry to reward history

Reward History should show:

* Reward name
* Cost
* Requested date
* Approved/Denied/Cancelled
* Adult action date
* Remaining points
