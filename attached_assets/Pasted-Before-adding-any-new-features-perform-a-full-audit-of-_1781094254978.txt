Before adding any new features, perform a full audit of the entire application.

Priority 1: Stability & Bug Fixes

Audit every game, screen, button, navigation route, profile function, score system, review screen, achievement system, and progress tracker.

Fix any issues where:

* A game fails to load
* A game throws an error
* A game opens a blank screen
* Scores are not saved
* Progress is not saved
* Review history is incomplete
* Buttons do not work
* Navigation breaks
* Recent Games entries are missing information

If a game is not fully built, either:

* Fully complete it, or
* Hide it until it is complete

No visible game should ever produce an error.

---

Priority 2: Age Appropriate Learning System

The current difficulty scaling is not strict enough.

The app must ensure every activity matches the child's age range.

Age Ranges:

* 3–4 Years
* 5–6 Years
* 7–8 Years

3–4 Years:
Focus on:

* Counting 1–5
* Colours
* Shapes
* Matching
* Animal recognition
* Memory skills
* Visual learning

Minimal reading.

5–6 Years:
Focus on:

* Counting 1–20
* Addition up to 10
* Picture subtraction
* Rhyming
* Missing letters
* Animal habitats
* Patterns
* Early reading

7–8 Years:
Focus on:

* Addition and subtraction up to 100
* Multiplication
* Division basics
* Times tables
* Money
* Time telling
* Reading comprehension
* Sentence building
* Spelling
* Science facts
* Logic puzzles
* Memory challenges
* Timed challenges

Remove baby-level questions from 7–8 content.

A 7–8 year old should never receive questions such as:

* Is 1 bigger than 2?
* Count 3 objects
* Identify basic colours

Difficulty must genuinely increase with age.

Priority 3: Fix Review Screen

Currently the review screen only shows:

Question 1

This is not enough.

Store and display:

* Full question text
* Any images used
* Dice shown
* Animal shown
* Shape shown
* Child's answer
* Correct answer
* Correct/Incorrect result
* Explanation

Example:

Question 1

Question:
Mia has 24 stickers and gets 18 more. How many does she have now?

Child Answer:
42

Correct Answer:
42

Result:
Correct

Explanation:
24 + 18 = 42

Every completed game should save its complete question history.
